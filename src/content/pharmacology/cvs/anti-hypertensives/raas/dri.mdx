# Renin Inhibitors

## Example

- `aliskiren` — the only clinically approved direct renin inhibitor.

> ↓ plasma renin activity → ↓ Ang II → vasodilation, ↓ sodium & water retention → ↓ BP.

## Mechanism of Action (MOA)

- **Directly inhibits renin**, the enzyme responsible for converting **angiotensinogen → angiotensin I**.
- This leads to ↓ formation of **angiotensin I** and subsequently ↓ **angiotensin II** and **aldosterone** levels.
- Acts **upstream** of both ACE inhibitors and ARBs.

## Pharmacodynamics

| Site of Action            | Effect                                                     |
| :------------------------ | :--------------------------------------------------------- |
| **Vasculature**           | ↓ Vasoconstriction → ↓ systemic vascular resistance        |
| **Kidney**                | ↓ Aldosterone → ↓ Na⁺ & H₂O reabsorption                   |
| **Heart/Vessels**         | ↓ Remodeling, ↓ sympathetic tone                           |
| **Plasma renin activity** | ↓ (distinct from ACEIs/ARBs, which cause reactive ↑ renin) |

## Pharmacokinetics

| Property           | Description                                       |
| :----------------- | :------------------------------------------------ |
| **Absorption**     | Poor (~2-5%), ↓ with fatty meals                  |
| **Peak effect**    | 1-3 hours                                         |
| **Half-life**      | ~24-40 hours → once daily                         |
| **Metabolism**     | Minimal hepatic; mainly biliary excretion (feces) |
| **Administration** | Oral (tablet)                                     |

## Indications

- **Essential hypertension** (monotherapy or in combination)
  - Particularly when ACEIs or ARBs are not tolerated.

_(No proven advantage over ACEIs/ARBs in heart failure or nephropathy.)_

## Adverse Effects

| System           | Effect                     | Notes                   |
| :--------------- | :------------------------- | :---------------------- |
| **CVS**          | Hypotension                | esp. if volume-depleted |
| **Renal**        | Hyperkalemia, ↑ creatinine | ↓ aldosterone           |
| **GI**           | Diarrhea                   | dose-dependent          |
| **Dermatologic** | Rash (rare)                | —                       |
| **Allergic**     | Angioedema (rare)          | similar to ACEIs        |

## Cautions

- Avoid **combining with ACEIs or ARBs** — ↑ risk of **renal failure, hypotension, hyperkalemia** (especially in diabetics).
- **Renal impairment**: monitor renal function closely.

## Contraindications

- **Pregnancy** (fetotoxic)
- **Bilateral renal artery stenosis**
- **Concurrent ACEI/ARB use in diabetics or renal impairment**
- **Severe renal dysfunction (eGFR < 30 mL/min/1.73m²)**

## Drug Interactions

- **NSAIDs** → ↓ antihypertensive effect, ↑ renal risk
- **K⁺ supplements / K⁺-sparing diuretics** → ↑ hyperkalemia risk
- **Cyclosporine, itraconazole** → ↑ `aliskiren` levels (avoid combination)

## Key Point Summary

| Feature               | **Renin Inhibitor (`aliskiren`)**                       |
| :-------------------- | :------------------------------------------------------ |
| **Target**            | Direct inhibition of renin (rate-limiting step in RAAS) |
| **Effect**            | ↓ Ang I, ↓ Ang II, ↓ aldosterone                        |
| **Bradykinin**        | Unaffected → **no cough**                               |
| **Reflex renin rise** | Suppressed                                              |
| **Main Use**          | Hypertension (alternative to ACEIs/ARBs)                |
| **Adverse Effects**   | Diarrhea, hyperkalemia, hypotension                     |
| **Contraindicated**   | Pregnancy, renal artery stenosis, ACEI/ARB combination  |
| **Special Note**      | No proven outcome benefit beyond ACEIs/ARBs             |

## 🩺 **RAAS Blockade Comparison**

| Feature                | **ACEIs**               | **ARBs**       | **Renin Inhibitors**           |
| :--------------------- | :---------------------- | :------------- | :----------------------------- |
| **Site of Action**     | ACE enzyme              | AT₁ receptor   | Renin enzyme                   |
| **Effect on Ang II**   | ↓ formation             | ↓ action       | ↓ formation                    |
| **Bradykinin**         | ↑ (→ cough, angioedema) | No effect      | No effect                      |
| **Cough/Angioedema**   | Common                  | Rare           | Rare                           |
| **Use**                | HTN, CHF, CKD, post-MI  | Same           | HTN only                       |
| **Contraindicated In** | Pregnancy, RAS          | Pregnancy, RAS | Pregnancy, RAS, ACEI/ARB combo |
| **Unique Feature**     | Cardioprotective        | No cough       | Upstream RAAS block            |
